# 대성쎌틱에너시스 비즈니스 맥락

## 회사 개요

| 항목 | 내용 |
|------|------|
| 회사명 | 대성쎌틱에너시스 (Daesung Celtic Enersys) |
| 그룹 | 대성그룹 계열사 |
| 사업 영역 | 가정용·상업용 보일러, 온수기, 냉난방 시스템, 에너지 솔루션 |
| 주요 시장 | 대한민국 (국내 중심) |
| 웹사이트 | https://www.celtic.co.kr |

---

## 산업 환경

### 시장 특성

- **계절성**: 냉난방 사업 특성상 **10월~3월(난방 시즌)** 이 핵심 판매 시즌
- **교체 수요 중심**: 신규 설치보다 기존 노후 보일러 교체 수요가 큰 비중 차지
- **건설 연동성**: 아파트·주거단지 신규 입주 물량에 따라 B2B 수주 변동
- **에너지 가격 민감도**: 가스비·에너지 요금 인상 시 고효율 제품 수요 증가

### 주요 경쟁사

| 경쟁사 | 특징 |
|--------|------|
| 경동나비엔 | 국내 1위, 해외(북미) 수출 강자, 콘덴싱 보일러 선도 |
| 린나이코리아 | 온수기·보일러 전문, 일본 브랜드 신뢰도 |
| 귀뚜라미 | 국내 시장 강자, 중저가 포지션, 넓은 AS 네트워크 |
| 대성쎌틱에너시스 | 프리미엄 콘덴싱 포지션, 대성그룹 브랜드 신뢰도 |

### 트렌드

- **친환경·탄소중립 규제 강화** → 고효율 콘덴싱 제품 수요 증가
- **스마트홈 연동** → IoT·앱 기반 원격 제어 보일러 수요 성장
- **에너지 절약 소비 심리** → 연비·효율 소구 마케팅 효과 증가
- **프리미엄 소비 양극화** → 고가 프리미엄 라인과 합리적 가격 라인 이원화 필요

## 고객 세그먼트

### B2C (개인 소비자)

| 세그먼트 | 특징 | 주요 니즈 |
|----------|------|-----------|
| 30~50대 주택 소유자 | 보일러 교체 주요 의사결정자 | 내구성, AS 신뢰도, 에너지 절감 |
| 신혼·신규 입주자 | 인테리어 관심 높음 | 디자인, 스마트 기능, 브랜드 이미지 |
| 고령층 가구 | 조작 편의성 중시 | 간편한 UI, 빠른 AS, 안전성 |

### B2B (기업 고객)

| 세그먼트 | 특징 |
|----------|------|
| 건설사·시행사 | 아파트·주거단지 대량 납품 (단가·납기 중시) |
| 인테리어·시공업체 | 제품 스펙·마진·AS 정책 중시 |
| 부동산 관리업체 | 임대주택·빌딩 유지보수 계약 |

## 마케팅 커뮤니케이션 전략

### 핵심 메시지

- **프리미엄 기술력**: "대성 블랙 콘덴싱 — 새로운 기준"
- **생활 밀착형 솔루션**: 환절기·겨울철 난방 관리 정보 콘텐츠 제공
- **신뢰와 안심**: 대성그룹의 오랜 기술 역사와 AS 네트워크 강조

### 주요 마케팅 채널

| 채널 | 활용 방식 |
|------|-----------|
| 자사 웹사이트 | 제품 소개, 매거진 콘텐츠, 온라인 상담·구매 |
| 디지털 광고 | 검색광고(네이버), SNS 광고(유튜브, 인스타그램) |
| 매거진 콘텐츠 | 계절별 난방 팁, 제품 활용 가이드 (자사 웹 매거진) |
| 오프라인 AS망 | 전국 서비스센터를 통한 브랜드 접점 |
| B2B 영업 | 건설사·시공업체 대상 직접 영업·전시회 |

### 콘텐츠 방향

- **계절 캠페인**: 환절기(봄·가을), 겨울 난방 시즌 집중 프로모션
- **에듀케이션 콘텐츠**: 보일러 관리법, 에너지 절약 팁 등 생활 정보
- **제품 출시 캠페인**: 신제품 론칭 시 프리미엄 비주얼 강조
- **AS·신뢰 콘텐츠**: 빠른 AS, 오랜 기술력 기반 안심 메시지

---

## 브랜드 커뮤니케이션 시 유의 사항

- 경쟁사 직접 비교 광고 지양 (품질·효율 수치는 공신력 있는 데이터 기반)
- 에너지 효율 수치 등 기술 스펙 표기 시 정확한 인증 데이터 사용 필수
- 환경(친환경, 탄소감축) 관련 표현은 실제 인증 기반으로만 사용
- 가격 프로모션 메시지는 시즌·재고 상황에 따라 별도 승인 후 사용

## 연간 마케팅 캘린더 (기준)

| 시기 | 주요 이벤트 | 마케팅 포커스 |
|------|------------|--------------|
| 1~2월 | 한파·난방 성수기 마무리 | 겨울 난방 솔루션, AS 캠페인 |
| 3~4월 | 봄 이사 시즌, 환절기 | 보일러 교체 프로모션, 환절기 관리 콘텐츠 |
| 5~8월 | 비수기 | 냉방 제품 소구, 브랜드 인지도 캠페인 |
| 9~10월 | 환절기, 난방 시즌 예열 | 난방 준비 캠페인, 신제품 론칭 타이밍 |
| 11~12월 | 난방 성수기 본격화 | 주력 제품 프로모션, 연말 특가 이벤트 |
