## ✦ 작성 가이드 & 체크리스트

### 어조 원칙
- 친근하고 따뜻한 말투 유지 ("~이에요", "~해요", "~드릴게요")
- 독자의 일상 상황 공감에서 시작 → 정보 제공 → 자연스러운 제품 연결
- 과도한 제품 홍보 없이, 정보 콘텐츠 비중 70% / 제품 소개 30%

### 구성 체크리스트
- [ ] 에피소드 번호 및 날짜 기입
- [ ] 이번 호 핵심 토픽 2개 설정
- [ ] 인트로에 독자 공감 질문 포함
- [ ] 문제점 3가지 체크리스트 (✅) 구성
- [ ] 해결책 섹션에 핵심 원칙 한 줄 강조
- [ ] 상황별 가이드 3가지 구성
- [ ] 생활 꿀팁 3가지 (①②③) 구성
- [ ] 제품 섹션에 수치/혜택 포함 (예: ~만원 절약, ~년 보증)
- [ ] 홈페이지 링크 삽입 (https://www.celtic.co.kr/)
- [ ] 클로징 슬로건 확인
- [ ] SNS 채널 링크 확인

### 이미지 방향
- 히어로: 계절감 있는 생활 공간 (따뜻한 베이지·화이트 톤)
- 섹션 이미지: 상황을 직관적으로 보여주는 생활 사진
- 제품 이미지: 인테리어 배경 + 제품 단독 컷 2분할
- 클로징: 제품 + 감성적 배경 (석양, 따뜻한 빛)

### 발행 주기 참고
- 월 1~2회 발행 (매거진 형식)
- 에피소드 번호 순차 증가
- 계절·이슈 캘린더에 맞춰 주제 설정
  - 3~4월: 환절기 난방 관리
  - 5~8월: 냉방/비수기 생활 팁
  - 9~10월: 난방 시즌 준비
  - 11~2월: 난방 효율·절약 집중
